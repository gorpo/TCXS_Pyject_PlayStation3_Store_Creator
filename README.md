<h3>OQUE ESTE PROGRAMA FAZ:</h3>
Este programa foi programado em Python3 e gerado um executavel do mesmo assim permitindo o usuario criar e editar lojas diretamente no seu console PlayStation3 e tambem compactar sua loja em PKG para distribuir para outros usuarios com consoles previamente exploitados/desbloqueados.<br>
- Escreve e cria lojas direteamente no console PlayStation3.<br>
- Escreve e cria sistema de IPTV diretamente no console PlayStation3.<br>
- Gera PKG para distribuição ou controle de versão.<br>
- Cria jogos de PlayStation3 PSP com um link na loja.<br>
- Cria jogos de PlayStation3 PS1 com um link na loja.<br>
- Cria jogos de PlayStation3 PS2 com um link na loja.<br>
- Cria jogos de PlayStation3 PS3 de 1 a 10 links podendo ser colocados jogos particionados na loja.<br>
- Cria sistema de Televisão com Animes no PlayStation3 com episodios de 1 a 10 links.<br>
- Cria sistema de Televisão com Desenhos no PlayStation3 com episodios de 1 a 10 links.<br>
- Cria sistema de Televisão com Filmes no PlayStation3 com episodios de 1 a 10 links.<br>
- Cria sistema de Televisão com Séries no PlayStation3 com episodios de 1 a 10 links.<br>
- Sobe todos arquivos para seu console PlayStation3 via FTP direto pelo programa.<br>
- Conferencia de arquivos XML PAI e XML FILHO.<br>
- Changelog detalhado para o usuario dos XML's criados no programa principal.<br>
- Changelog de upload via FTP detalhado via terminal.<br>
- Changelog em cada XML criado com sistema de data | hora | nome de usuario.<br>
- Inserção e conferencia do endereço de IP do console PlayStation3.<br>
- Sistema de login e senha.<br>
- Limpeza de cache automatizada.<br>
- Senha disponivel apenas para compradores.<br>
